# Scripts

This directory contains automation scripts for the AI assistant framework.

## Purpose

Place shell scripts, Node.js scripts, or other automation tools here that support the framework workflows.

## Examples

- `init.sh` - Project initialization helper
- `validate.js` - Custom validation runner
- `migrate.sh` - Framework upgrade migration

## Usage

Scripts should be referenced from workflows or run manually as needed.
