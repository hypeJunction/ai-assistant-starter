# Test Plans

This directory contains test plan templates and documentation for the AI assistant framework.

## Purpose

Store reusable test plan templates that can be referenced when creating tests for specific components or features.

## Structure

```
test-plans/
├── component-template.md    # Template for component tests
├── integration-template.md  # Template for integration tests
└── e2e-template.md          # Template for end-to-end tests
```

## Usage

Reference these templates from the testing guidelines in `domains/testing.instructions.md`.

---

**Note:** This directory is currently empty. Add test plan templates as needed for your project.
